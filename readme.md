# SQL_Maker
-------
# 손쉽게 csv, xlsx파일을 저장하자
- csv, xlsx를 손쉽게 Database table로 제작하자!
- database table에 손쉽게 데이터를 삽입하자!
- MySQL 으로 데이터를 원하는대로 조회하고 csv파일로 저장하자!

------
## 필요한 모듈
- python 3.8.1
- pandas
- PyQt
- mysql
- xlrd
